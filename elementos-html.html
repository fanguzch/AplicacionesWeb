Instrucciones

El siguiente enlace contiene la lista de elementos de HTML soportados por la mayoría de navegadores: HTML elements reference.

  • Cree un documento HTML con la estructura básica de un documento de HTML5 (visto en clase anterior).
  • Del enlace anterior, seleccione 10 elementos de cualquiera de las siguientes categorías (Content sectioning, Text content, Inline text semantics, Image and multimedia, Table content, Forms).
  • En el documento HTML, utilice los 10 elementos seleccionados. Cada uno de los elementos debe estar separado por un título con el siguiente formato: Utilizando la etiqueta [NOMBRE]
  • Incluya este archivo en el repositorio creado anteriormente en la tarea de repaso de Git.
  • Suba el documento al repositorio remoto.
