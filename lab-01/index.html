Información básica

Creadores

Foundation fue desarrollado por Zurb, una compañía de diseño de productos digitales.

Fecha de publicación

Foundation fue lanzado por primera vez en 2011.

Enlace a GitHub

Repositorio de Foundation en GitHub

Fun fact

Foundation fue el primer framework CSS en ser completamente responsive por defecto.

Casos de uso

Web #1

Adobe usa Foundation para su diseño adaptable.

Web #2

eBay implementa Foundation en partes de su interfaz.

Web #3

Mozilla ha usado Foundation en proyectos internos.

Pasos para la instalación

Para instalar Foundation, puedes usar npm:

npm install foundation-sites
